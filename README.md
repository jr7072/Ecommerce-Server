## Description

Backend for ecommerce site. Will include a REST API and  Relational Database.

## Technologies

+ Node.js (17.8.0)
+ Express.js (4.17.3)
+ Postgresql (14.2)

## Setup

1. Clone repository
```
    $ git clone https://github.com/jr7072/Ecommerce-Server.git
```

2. start postgresql service

3. Run Server (runs locally on port 4000)

```
   $ node server.js
```
